COLORS AND DESIGN

This is an API created in python with the flask framework using a JSON file with {% if colors%} over {{ colors}}{%else %} more than 25,000{% endif %} colors! Created during a 6 hr hackathon by Paula Bannerman.

Routes:
  • /colors/full/ is for seeing the page of a listing of 1000 colors (pagination)
  • /colors/search/ is for finding the specific name of a color
  • /colors/id/ find a color by id
  • /colors/digit/ to find a color by the first letter or digit
  • /colors/snippet/ find a color by a snippet of the name
  • /colors/hex/snippet/ find a color by a snippet of the hex code
  • /colors/hex/code/ find a color by hex code
